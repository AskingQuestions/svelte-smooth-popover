# svelte-smooth-popover

A robust, unstyled popover component for Svelte with a customizable smooth/sharp caret.

## Installation

```bash
npm i svelte-smooth-popover
```

## Usage

```html
<button>
	Example
	<Popover>
		<div class="shadow-sm bg-white">Popover content</div>
	</Popover>
</button>
```
